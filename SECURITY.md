# Security policy

## Supported versions

Security fixes are provided for the latest published version of `@yard/cli`.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use [GitHub private vulnerability reporting](https://github.com/Stupremee/yard/security/advisories/new) and include:

- A description of the issue and its impact
- Reproduction steps or a proof of concept
- Affected versions and environments
- Any suggested mitigation, if available

Please allow maintainers a reasonable amount of time to investigate and release a fix before public disclosure.
